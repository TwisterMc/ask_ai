Settings

You can provide your own Pollinations API key here. It will be stored in your browser's local storage only and sent with requests from your browser to this server so the server can forward it to the Pollinations API for you.

How to get an API key

  1. Sign in at pollinations.ai.
  2. Open your account or API settings and create a new API token.
  3. Copy the token and paste it into the API Key field above, then click Save.
